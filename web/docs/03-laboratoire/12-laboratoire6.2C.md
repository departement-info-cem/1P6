# 🧪 Labo 6.2C – 🔴 Devine nombre

---

## 🎯 Objectifs 
* Se familiariser avec Windows Forms et les classes
* Créer une classe
* Instancier et utiliser un objet
* Créer un formulaire lié à un objet

---

## 🛠️ Instructions

Vous devez écrire un petit jeu qui permet de deviner un nombre entre deux valeurs sélectionnées. Le jeu compte le nombre de tentatives effectuées.

- Vous aurez à créer une classe **`Jeu`** qui s'occupe de trouver un entier aléatoire entre deux valeurs.
- Vous devez avoir un minimum et un maximum par défaut.
- Le jeu doit compter le nombre de tentatives.
- Vous aurez besoin d'une boîte de texte pour le chiffre qui sera essayé.
- Un bouton **Essayer** qui demande au **`Jeu`** si le chiffre essayé est gagnant.
    - Si c'est le cas, afficher un **`MessageBox`** qui félicite l'utilisateur d'avoir trouvé!
    - Dans tous les cas, le jeu devrait augment le nombre d'essaie de 1.
- Un bouton qui permet de **réinitialiser** le jeu selon le minimum et le maximum spécifié.

:::warning
Pour cet exercice, tout ce qui se rapporte aux règles du jeu doit se trouver dans la classe **`Jeu`**! 
:::
